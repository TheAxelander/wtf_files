# tmux Panes

| Key Bindings | Description |
| --- | --- |
| `CTRL + B %` | Vertical split |
| `CTRL + B "` | Horizontal split |
| `CTRL + B SPACE` | Cycle through layouts |
| `CTRL + B →` | Move to right pane |
| `CTRL + B ←` | Move to left pane |
| `CTRL + B ↓` | Move down to pane |
| `CTRL + B ↑` | Move up to pane |
| `CTRL + B O` | Go to next pane |
| `CTRL + B ;` | Go to last active pane |
| `CTRL + B {` or `}` | Swap pane with previous/next |
| `CTRL + B !` | Break pane into new window |
| `CTRL + B X` | Kill pane |
| `CTRL + B :` | Open command prompt |
| `CTRL + B z` | Zoom/unzoom current pane |

## Copy Mode & Selection

| Key Bindings | Description |
| --- | --- |
| `CTRL + B [` | Enter copy mode |
| `CTRL + Space` | Start selection (emacs mode) |
| `Enter` | Copy selection to tmux buffer |
| `CTRL + B ]` | Paste from tmux buffer |
