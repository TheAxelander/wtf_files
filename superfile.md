# Superfile

## Navigation

| Shortcut | Description |
| --- | --- |
| `s`  | focus on the sidebar |
| `p` | focus on the processes |
| `m` | focus on the metadata |
| `:` | open command execution bar |
| `f` | show or hide the preview window |
| `F` | hide or show all footer panel |
| `n` | create file panels |
| `w` | close the focused file panel |
| `TAB` | move through multiple file panels |
| `P` | pin folder to sidebar |
| `o` | sort options menu |

## File Operation

| Shortcut | Description |
| --- | --- |
| `v` | toggle between selection mode and normal mode |
| `CTRL` + `n` | create new file |
| `CTRL` + `c` | copy file |
| `CTRL` + `v` | paste file |
| `CTRL` + `x` | cut file |
| `CTRL` + `r` | rename file | 
| `CTRL`+ `d` | delete file |
| `e` | open file with editor |
