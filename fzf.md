# fzf

| Command | Description |
| --- | --- |
| `nano **` + `TAB` | Search file for editing via fzf |
| `ALT + C` | Fuzzy find all subdirectories of the working directory and cd into it |
| `STRG + R` | Search in history via fzf |
| `ssh` + `STRG + R` | Search for ssh command in history via fzf |
| `<command>` + `STRG + T` | Fuzzy find file and append result to command |
| `sudo !!` | Run last command with sudo |
