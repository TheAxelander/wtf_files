# Kitty

| Key Bindings | Description |
| --- | --- |
| `CTRL + SHIFT F5` | Reload config |
| `CTRL + SHIFT ENTER` | New Window |
| `CTRL + SHIFT l` | Cycle through Window layout |
| `F11` | Toggle Zoom |
| `CTRL + SHIFT F7` | Visually focus window |
| `CTRL + SHIFT F8` | Visually swap window |
| `CTRL + SHIFT up/down` | Focus window navigation |
| `CTRL + SHIFT t` | New Tab |
| `CTRL + SHIFT ,/.` | Move tab |
| `CTRL + SHIFT left/right` | Tab navigation |
| `CTRL + SHIFT + ALT t` | Set tab title |
